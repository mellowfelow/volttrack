# VoltTrack

America's electric dirt bike store — React + Next.js (App Router), deployed on Vercel. Built with WebForge.

## Stack
- Next.js 14 (App Router), React 18
- Single source of truth: `src/config/site.js`
- Agent-ready files + `vercel.json` generated by `scripts/gen-agent-files.mjs`

## Commands
```bash
npm install
npm run dev          # local dev
npm run build        # prebuild generates agent files, then next build (Vercel target)
npm run crosscheck   # pre-ship checks (must pass before push)
npm run images       # normalise product photos (placeholder fallback if none)
npm run build:static # Cloudflare static export (out/) — not the primary target
```

## Deploy (Vercel)
1. Create an empty GitHub repo, then:
   ```bash
   git remote add origin <repo-url>
   git push -u origin main
   ```
2. Vercel → Add New → Project → import the repo → **Framework Preset: Next.js** → Deploy.
3. `vercel.json` supplies headers, redirects and the agent-ready `Link` header.

## Connecting a domain later
Set `SITE.domain` in `src/config/site.js`, rebuild, push. Never find-and-replace a domain across files.

## Placeholders to set before launch
| Placeholder | File | Effect until set |
|---|---|---|
| `FORMS.web3formsKey` | `src/config/site.js` | Contact form redirects to thank-you but emails nowhere |
| `CHAT` tawk `PROPERTY_ID/WIDGET_ID` | `src/config/site.js` | Tawk widget not injected (email link still works) |
| `SITE.gscVerification` | `src/config/site.js` | No Google Search Console meta tag |

## Content source
`_source/` holds the original hand-built static site. It is the **content source only**, is
gitignored, and is never deployed.
